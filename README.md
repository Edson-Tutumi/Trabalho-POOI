# Trabalho-POOI

Um jogo por texto, onde você escolhe seu personagem e parte em uma aventura escolhendo os passos do seu heroi, cada heroi tem seus status e estes influenciam na possibilidade de ter sorte ou azar quando se passa por um desafio, mas cuidado, alguns desafios nem sempre são vencíveis

Primeiro é necessário ter o java instalado, se não possuir baixe-o aqui: https://www.oracle.com/java/technologies/downloads/ e faça as configurações necessárias.

Para executar o jogo abra a pasta onde o jogo foi extraido, clique na barra de pesquisa logo acima (na barra de pesquisa se tem o caminho da pasta, apague ele e apenas tenha na barra o que é dito a seguir), digite 'cmd' e pressione enter, o prompt de comando vai abrir, agora copie, cole e de enter no seguinte comando: javac CenaPrincipal.java, se tudo ocorrer bem e o prompt não mostrar nenhuma mensagem de erro, copie, cole e de enter nesse outro comando: java CenaPrincipal, e assim o jogo começara

Alguns dos conceitos aplicados são a herança, abstração, polimorfismo, interfaces, tratamento de exceções além de estruturas de dados como hashmaps e arraylists

O item criativo foi basicamente a forma como a qual são criados os estagio, utilizando hashmaps e um contador, para encontrar suas diferentes partes a partir do valor no contador quando tal estagio foi criado
